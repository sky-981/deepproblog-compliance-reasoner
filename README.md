# DeepProbLog Compliance Reasoner

A neuro-symbolic GDPR compliance reasoner: an LLM extracts probabilistic facts from a messy scenario, a ProbLog rulebook reasons over them, and the system returns a probability of compliance together with a proof tree that shows exactly which rules and facts drove the verdict. When it is not confident enough, it abstains and asks for human review instead of guessing.

Work in progress.

> Disclaimer: this is a reasoning demo, not legal advice.
